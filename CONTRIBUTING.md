# Contributing to Krishi AI 4.0 🌾

First off, thank you for considering contributing to Krishi AI! It's people like you that make digital agriculture better for everyone.

## 🧪 Scientific Integrity
As this project is grounded in **BARI**, **BRRI**, and **BARC** standards, all diagnostic logic or advisory changes must:
1. Cite an official government source or scientific paper.
2. Be peer-reviewed by an agronomist on our team.
3. Be tested against our standard LCC and Pest datasets.

## 🛠️ Development Workflow
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests!
3. Ensure the test suite passes (`npm run build`).
4. Submit a Pull Request with a clear description of the improvement.

## 📜 Code of Conduct
We are committed to providing a welcoming and inspiring community for all. Please be respectful and professional in all interactions.

## 🏗️ Technical Stack
- **Frontend**: React 19 / Vite
- **AI**: Gemini 3 Flash & Qwen-2.5-VL
- **Backend**: FastAPI / Python
- **Database**: Supabase
